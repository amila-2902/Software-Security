Follow this steps to run this web application.
1.Extract this MS20902902.rar folder.
2.Copy this Authentication folder to your WAMP or XAMP server.(Respectivey WAMP WWW folder and XAMP htdocs folder).
3.Run WAMP or XAMP server.
4.Open you favourite browser. Type in address bar "Localhost/Authentication/login.php" .
5.Then click on Log-in with Google red button.
6.Select your sliit email address to authentication and log in.
